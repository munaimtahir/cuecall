# Cursor Prompt — Queue Manager MVP

Build the full project structure for an Android app named `CueCall` using Kotlin + Jetpack Compose.

## Objective
Create a clean, production-ready Android app scaffold for a clinic queue management system with these modules:
- Reception
- Counter
- Display
- Settings
- Printer
- Data
- Domain

## Required stack
- Kotlin
- Compose
- MVVM
- Clean architecture
- Hilt
- Room
- Firebase Firestore
- Navigation Compose
- Coroutines / Flow

## Required output
1. Complete Android project scaffold
2. Package structure
3. Model classes
4. Repository interfaces + initial implementations
5. Compose navigation shell
6. Placeholder UI screens with proper state wiring
7. Printer abstraction layer
8. README
9. TODO checklist at end

## Rules
- no fake business logic shortcuts
- no giant monolithic activity
- structure for future scaling
- use meaningful package names
- include comments only where actually useful
- keep UI simple and usable

## Required screens
- Splash / Mode selection
- Reception screen
- Counter screen
- Display screen
- Settings screen
- Service management screen
- Printer settings screen

## Must create
- Firestore-ready token entity
- Room entities/DAO
- queue repository
- token generation use case
- call next use case
- complete token use case
- printer manager interface

## End with
A strict TODO checklist covering missing implementation pieces.
