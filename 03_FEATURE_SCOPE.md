# MVP Feature Scope

## Included in MVP
- create token
- print token via Bluetooth printer
- multi-service support
- separate token prefix by service
- live queue board on Android display mode
- call next token
- recall current token
- skip token
- complete token
- token history for current day
- daily queue reset
- printer pairing/settings
- local cache

## Explicitly deferred
- patient registration module
- appointment scheduling
- billing/payment
- WhatsApp integration
- SMS integration
- multi-branch enterprise admin
- web portal
- advanced analytics
- multilingual voice engine
- cloud reporting dashboards
