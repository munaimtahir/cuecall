# Codex Prompt — Bring Up, Verify, and Smoke Test CueCall Android Project

You are given an Android project for CueCall.

## Your job
Perform a full engineering validation pass and bring the project to a runnable, verified state.

## Tasks
1. Inspect Gradle/build configuration
2. Fix dependency/version issues
3. Verify Hilt, Room, Compose, Firebase setup consistency
4. Ensure project builds cleanly
5. Run available tests
6. Add or fix missing seed/demo data path
7. Validate navigation flows compile
8. Check printer abstraction compiles even if physical printer unavailable
9. Produce a concise PASS/FAIL report

## Important
- Do not redesign architecture unless necessary
- Fix small/systemic issues directly
- Keep scope within MVP
- Do not add non-MVP product features

## Deliverables
- list of fixes made
- build status
- test status
- remaining blockers
- final TODO checklist
